# Zlatá trefa – audioguide

Toto je GitHub Pages verze s klasickými relativními cestami. Nahrajte do kořene repozitáře:

- `index.html`
- složku `assets` včetně podsložek `audio` a `panels`
- `.nojekyll`

Po nahrání proveďte v prohlížeči tvrdé obnovení stránky: `Ctrl + Shift + R`.

Pokud GitHub Pages publikuje z podsložky `/docs`, vložte všechny tyto soubory právě do `/docs`.
